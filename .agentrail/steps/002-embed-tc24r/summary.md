Embedded tc24r C compiler (preprocess/lex/parse/codegen) and cor24-rs assembler+emulator as WASM dependencies. Created compiler.rs with full pipeline: C source → assembly → machine code → execution. Added standalone feature gate to cor24-emulator to prevent its wasm_bindgen(start) from conflicting. Three-pane UI shows C source, generated assembly, and execution output (UART Hi!, registers r0=0x2a, LEDs). Verified end-to-end with trunk build and Playwright testing on port 9101.
